# Online Store (Angular)

This project is a simple **Online Store web application** built with **Angular** as part of **Lab 04 - Task 2**.

The application displays a list of products in a grid layout.  
Each product card contains an image, name, description, price, rating, and share buttons.

---

## Features

- Product list page (grid layout)
- Product card component
- Product images gallery (multiple images)
- Product rating (stars)
- Share buttons:
  - WhatsApp
  - Telegram
- Link to Kaspi product page

---

## 🛠 Technologies Used

- Angular
- TypeScript
- HTML / CSS
- Angular Standalone Components

---
